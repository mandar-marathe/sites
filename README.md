# About this Archive
This repository contains an archival snapshot of https://rhetorical-density.com, taken on Monday 24 November 2025.  

The purpose of this archive is to timestamp and preserve the original formulation and presentation of the concept Rhetorical Density, a metric developed by [Dr Mandar Marathe](https://marathe.org) (mandar@marathe.org) for quantitative analysis of rhetorical devices in texts.

* PDF screenshot of the site
* SingleFile HTML snapshot
* Extracted source code.
